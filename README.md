# Smart Learning Dashboard

## Description

The Smart Learning Dashboard is a responsive frontend web application developed using HTML5, CSS3, and JavaScript. It helps students manage assignments, attendance, and notes in one place.

---

## Features

- Dashboard Statistics
- Assignment Management
- Attendance Tracker
- Notes Manager
- Search Functionality
- Dark Mode
- Local Storage
- Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- DOM Manipulation
